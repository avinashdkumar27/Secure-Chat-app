About Secure Chat App

What is this?

Secure Chat App uses Public-Private Key pairs to ensure messages are only readable by the intended user.

How it works:

  1. Key Generation: When you register, you get a Public ID (shareable) and Private ID (secret).
  2. Encryption: When you send a message, it's encrypted with the receiver's Public ID.
  3. Decryption: Only the receiver can decrypt it using their Private ID.
  4. Security: Even the server cannot read your messages!
